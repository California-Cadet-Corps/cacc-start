# 🎒 Student Developer Guide — Start Here!

Welcome to the **California Cadet Corps** dev team! 🎉

This is a real website that real people use: **[start.cacadets.org](https://start.cacadets.org)**.
You're going to help build it — for real. The code you write (after it's checked
and approved) goes live on the internet. How cool is that?

This guide is written for **you**, even if you've never coded before. Take it one
step at a time. It's totally normal to feel confused at first — every developer
did. Read slowly, copy the commands exactly, and ask for help when you're stuck.

---

## 🗺️ The big picture (read this first!)

Here's the whole journey, start to finish:

```
   💡 You have an idea
        │
   📋 Plan it      ← decide what it should do (Product Management)
        │
   🎨 Design it    ← decide how it looks  (UI/UX)
        │
   💻 Build it     ← write the code (with help from Claude!)
        │
   ✅ Test it      ← make sure it works
        │
   📤 Pull Request ← ask the team to review your work
        │
   👀 Review       ← a teacher/mentor checks it and approves
        │
   🚀 Deploy       ← it goes LIVE on the website automatically!
```

You'll learn every one of these steps in the guides below.

## 🆘 Stuck? That's normal!

- **Read the error message.** Computers usually tell you what's wrong. Copy it and
  read it slowly.
- **Ask Claude.** Paste the error and ask "What does this mean and how do I fix it?"
- **Ask a teammate or mentor.** Use the
  [Discussions tab](https://github.com/California-Cadet-Corps/cacc-start/discussions).
- **Take a break.** Seriously. Walk away for 5 minutes. Bugs are easier after a snack. 🍎

## 🌟 The Golden Rules

1. **You can't break the live website.** Your changes are checked and approved
   before anything goes live. Experiment freely!
2. **Small steps win.** Change one thing, test it, then change the next.
3. **Copy commands exactly** — spaces, dashes, and capital letters all matter.
4. **Ask questions.** There are no dumb questions on this team.
5. **Have fun.** You're building something real. Enjoy it! 🚀
