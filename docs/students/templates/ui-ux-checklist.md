# 🎨 UI/UX Checklist

> Run through this before you call a design "done." Check each box. If you can't
> check it, fix it first!

---

## Looks good (UI)

- [ ] **One clear main thing** — the most important item is the biggest / most obvious.
- [ ] **Readable text** — good contrast (dark on light, or light on dark). No light-gray-on-white!
- [ ] **Consistent colors** — I used only 2–3 colors, and the same ones everywhere.
- [ ] **Consistent fonts** — same font for body text across the page.
- [ ] **Buttons look like buttons** — people can tell they're clickable.
- [ ] **Enough space** — things aren't crowded; sections can "breathe."

## Easy to use (UX)

- [ ] **Obvious purpose** — a stranger can tell what the page is for in 5 seconds.
- [ ] **Clear links** — link text says where it goes ("Join the Cadets," not "Click here").
- [ ] **Feedback** — buttons react when hovered/tapped; actions show a result message.
- [ ] **Friendly errors** — if something goes wrong, the message is clear and kind.

## Works everywhere

- [ ] **Phone-friendly** — I previewed it at phone size (F12 → phone icon) and it works.
- [ ] **Big-screen-friendly** — it still looks good on a large monitor.
- [ ] **Tap targets** — buttons are big enough to tap with a thumb.

## Accessibility (works for everyone) ♿

- [ ] **Not color-only** — I don't rely on color alone to show meaning.
- [ ] **Images have alt text** — every image has a short description in its `alt`.
- [ ] **Logical order** — content reads top-to-bottom in a sensible order.
